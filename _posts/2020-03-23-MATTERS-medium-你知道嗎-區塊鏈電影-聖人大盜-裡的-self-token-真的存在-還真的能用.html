真實上鏈發行的虛擬貨幣 — SELF TOKEN

徐嘉凱導演曾拍攝酒吧網劇《私室（ SELF Bar）》後，依著劇中描繪開了一間同名酒吧「SELF 私室」，生意興隆，甚至獲時尚雜誌 GQ 選為台北 20 間最好酒吧！而這一次《聖人大盜》電影拍攝前，徐導找上 寶博士（dAb）葛如鈞 時，就說了：希望拍攝一部區塊鏈主題電影，也要在真實世界中創辦一間和劇中一模一樣的區塊鏈新創公司，甚至推出相同名稱的產品「SELF TOKEN」！

SELF TOKEN 是什麼？為何要買 SELF TOKEN？

如果你和我一樣看過電影的話（以下無劇透），應該會知道，SELF TOKEN 就是一個可以拿來在超市裡結帳、可以和音樂公司老闆簽約鏈上賣音樂，也可以透過透明公正的玻璃杯（帳本）改變這個不公不義世界的重要武器（太入戲…XD）。

總之，電影裡所描述的 SELF TOKEN 發行募資真的存在過，SELF TOKEN 也真的有在區塊鏈上發行，甚至已經在 BitAsset (BA) 交易所上面可以進行交易，而且是一個「真的能夠用，有應用場景的虛擬貨幣」。
