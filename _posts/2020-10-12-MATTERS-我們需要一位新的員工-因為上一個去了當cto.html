LikeCoin 和 Matters 自去年深度合作後，就開墾了不少板塊，然而播下的種子多了，我們亦需要更多「農夫」灌溉，因此我們希望邀請更多小夥伴，加入 LikeCoin 團隊，築建共和國的一磚一瓦。

關於LikeCoin

LikeCoin 是基於 Cosmos SDK 構建的公共區塊鏈，它是一個去中心化的發布基礎架構，可為內容創建者和讀者提供去中心化的檔案，獎勵，策展和治理功能。 

我們目前正在開發 Liker Land 手機應用程式 ，Liker Land 建立在 LikeCoin 的區塊鏈技術之上，是一個面向讀者和創作者的手機應用程式。

招全端開發人員 Full Stack Developer

我們的技術棧：

前端：JavaScript / TypeScript，Vue，React Native

後端：JavaScript / TypeScript，Node.js，Firebase SDK

區塊鏈：Cosmos SDK，Tendermint

基礎架構：GCP，Firebase，K8，Redis，Elasticsearch，BigQuery，DataFlow

協作：Slack，Notion

  • 開發網頁和應用程序以完善 LikeCoin 生態系統
  • 為我們的開發人員社區開發 Web 和區塊鏈 API 和 SDK
  • 建立用於日常例行和分析的數據管道
