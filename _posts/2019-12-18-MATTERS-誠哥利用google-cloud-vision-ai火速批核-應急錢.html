李嘉誠基金會早前推出「應急錢」計劃，總共接獲四萬三千宗申請，冀幫助飲食業及零售中小企渡過時艱。為什麼基金會可以在三星期內完成審批所有申請？誠哥眼光非常獨到，團隊利用Google Cloud Vision AI成功火速完成十億港元的「應急錢」申請。除了給予誠哥掌聲外，建議花點時間了解一下他所選的人工智能平台有什麼功能。

先介紹是次計畫所用的技術「光學字符識別技術（OCR）」。OCR技術是利用光學設備（包括手機及其他攝像鏡頭）去捕獲圖像，只要圖像中內含文字，就能辨認出來。例如，在工作會議中，要記錄白板上寫上的文字，只需拍照並輸入到OCR軟件．文字就能提取出來，非常方便。

相信你已經聯想到李嘉誠基金會就是利用Google Cloud Vision AI的OCR功能來加快審批申請，系統自動偵測、摘錄和分析圖像中的相關文字，並利用OCR功能準確地自動輸入所偵測到的文字。Google雲端香港主管Lucy Werner表示，Google Cloud Vision AI可以在五秒辨認每個圖像檔中的文字，雖然不是即時轉換，但亦可大大節省大量人手處理和核對申請資料時間。

其實Google Cloud Vision AI十分強大，除了OCR 文字辨識外，還有這些非常實用的功能：

一、標籤偵測：系統會把圖中所有元素提取出來，並各自賦予一個分數，代表這個元素在圖中「真正」出現的信心程度。

二、臉部偵測：系統可以偵測圖中多個臉孔，並可針對圖中的每張臉，個別指出臉部的喜怒哀樂、特徵、位置、是否有戴眼鏡、是否模糊等等。

三、標誌或地標偵測：此功能可偵測出圖中是否含有著名商標或地標。

Google在2018年Google Cloud NEXT大會公佈正式推出一系列機器學習（Machine Learning）產品，命名為Cloud AutoML平台，涵蓋了圖像辨識（Vision）、翻譯（Translate）及自然語言處理（Natural Language）三大範疇。Cloud AutoML預計能解決很多產業問題，包括娛樂、媒體、醫療、保險產業等等，讓各企業能夠非常輕鬆地導入AI到自家產品。

相比起誠哥出錢又出力，獲批受助單位在短時間內收到「應急錢」，政府真是每次慢人一拍（甚至乎慢很多拍）。關愛共享計劃的四千元資助耗用大量人力物力，花了九個月還未完全完成審批過程。非常希望政府高官能與時並進，利用科技為市民帶來方便。
