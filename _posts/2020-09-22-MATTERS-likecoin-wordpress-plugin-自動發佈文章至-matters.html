若你用 WordPress 網站發佈文章，同時又想把文章發佈至 Matters，令文章內容能永久儲存在「星際檔案系統」 (IPFS) ，增加曝光並感受 Matters 上的互動社群氣氛，現在可使用 LikeCoin plugin 的新功能把文章自動發佈到 Matters 了。

設定 WordPress LikeCoin Plugin

首先當然需要已註冊 Liker ID 及 Matters 帳號，並把 Liker ID 及 Matters 帳號綁定。會讀此文的我假設都已是 LikeCoin 及 Matters 的用戶了，註冊流程便不赘。

接著，在你的 WordPress 網站的管理員介面，下載並安裝 LikeCoin plugin。

安裝完成並設定好 Liker ID 後，在 WordPress 管理介面左方菜單選 LikeCoin -> Publish Setting (beta)。

右方會出現一頁很簡單的表格。先填上你的 Matters 帳號資料（電郵及密碼），點 “login” 鍵，在表格「Matters connection status」部份確認已成功登錄 Matters 。頁底有兩個選項： “auto save draft to Matters” 要勾選，但 “auto publish to Matters” 建議別打勾，因為在 Matters 上文章一經刋登，便不能回頭修改，在使用 WordPress 這應用場景中，在 WordPress save draft 便立即發佈也不合道理。

設定好參數後，記得在頁底點 “Confirm” 鍵儲存改動。

自動同步 Matters 草稿

設定好plugin 後，往後每次在 WordPress 網站儲存一篇草稿、或更新/發佈一篇文章，對應的 Matters 帳號下也會生成一篇新的草稿。多次更新 WordPress 的文章，會反映到對應的 Matters 草稿，不用怕會生成了多餘的版本。

Feature Image 暫時無法同步到 Matters，有待改善；但內文中的圖片可同步。內嵌的 Youtube 影片（如本文的例子）也無法同步。

同步內容時，LikeCoin plugin 會把所有內文以外外掛產生的內容也加到 Matters 去，例如 “Related post“；要在 Matters 把這些多出的資料手動刪掉。

有了這新功能，以後一文多貼變得簡單得多了。
